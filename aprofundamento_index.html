Aprofundamento

Como converter um número para o sistema octal ou o oposto?

Sistema octal para decimal:

Exemplo: 75 na base 8

  1. Identifique o número em octal: 75
  2. Desdobre o número em suas partes 7 + 5
  3. Multiplique cada dígito pelo valor posicional correspondente: 7 x 8^1 + 5 x 8^0 = 54 + 5
  4. Some os resultados: 61

Sistema decimal para octal:

  1. Divida o número por 8
  2. Registre o resto da divisão: O resto de cada divisão será um dígito do número octal final, começando do último dígito até o primeiro
  3. Repita o processo: Continue dividindo o resultado da divisão anterior até que o quociente seja 0
  4. Leia os dígitos do resultado: Os restos que você registrou, lidos da última divisão para a primeira, representam o número na base octal

Exemplo: 61 em decimal é equivalente a 75 em octal.

Qual a importância da aplicação do sistema octal na computação?

  • Como abreviação de binário;
  • Representação de palavras de 16, 32 ou 64 bits;
  • Descrições de codificação;
  • Permissões de acesso a arquivos e cálculos;
  • Telas digitais;
  • Representação gráfica de cadeias de bytes.
