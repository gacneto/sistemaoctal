Introdução

O que é o sistema octal?

O sistema numérico octal é uma base numérica que utiliza oito símbolos para representar valores. Esses símbolos são os dígitos de 0 a 7. Similar ao sistema decimal, onde temos 10 símbolos (0 a 9), o sistema octal tem a vantagem de ser usado em situações onde a representação em base 8 é conveniente, especialmente em computação e eletrônica. Cada posição em um número octal representa uma potência de 8. Por exemplo, no número 247 em octal, o '2' representa 2x8², o '4' representa 4x8¹, e o '7' representa 7x8⁰. Esse sistema é utilizado em várias áreas da computação, especialmente em programação de sistemas e operações de baixo nível devido à sua relação com o sistema binário.

A seguir é apresentada a representação geral de um número octal:

num=ABC

ABC = A x 8^2 + B x 8^1 + C x 8^0

exemplos:

147 = 1 * 8^2 + 4 * 8^1 + 7 * 8^0
