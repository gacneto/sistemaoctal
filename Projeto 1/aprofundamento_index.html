Aprofundamento

Como converter um número para o sistema octal ou o oposto?

Sistema octal para decimal:

Exemplo: 75 na base 8

Passos:

Identifique o número em octal: 75

Desdobre o número em suas partes 7 + 5

Multiplique cada dígito pelo valor posicional correspondente:7 x 8^1 + 5 x 8^0 = 54 + 5

Some os resultados: 61

Sistema decimal para octal:

Divida o numero por 8

Registre o resto da divisão: O resto de cada divisão vai ser um dígito do número octal final, começando do último dígito até o primeiro.

Repita o processo: Continue dividindo o resultado da divisão anterior até que o quociente seja 0.

Leia os dígitos do resultado: Os restos que você registrou, lidos da última divisão para a primeira, representam o número na base octal.

61/8 = 7 com resto 5

7/8 = 0 com resto 7

Lemos os restos da última divisão para a primeira, que são 75. Portanto, 61 em decimal é equivalente a 75 em octal.

Portanto, para converter um número decimal para octal, basta dividir sucessivamente por 8 e ler os restos ao contrário.
